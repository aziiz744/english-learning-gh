# 📱 دليل تشغيل التطبيق على ويندوز (خطوة بخطوة)

هذا الدليل يشرح كيف تحوّل الموقع لتطبيق وتشغّله على جهازك.
اتبع الخطوات بالترتيب تماماً.

---

## ✅ المتطلّبات (مرّة واحدة فقط)

### 1. ثبّت Node.js
- حمّله من: https://nodejs.org (اختر النسخة LTS)
- بعد التثبيت، تأكّد بفتح موجّه الأوامر (CMD) واكتب:
  ```
  node --version
  ```
  لازم يطلع رقم نسخة (مثل v20.x.x)

### 2. حمّل ملفات المشروع
- فُكّ ضغط ملف `English-Journey-GitHub-Pages.zip`
- ستجد مجلد `english-learning-gh`

---

## 🚀 التشغيل (في كل مرة)

افتح موجّه الأوامر (CMD) **داخل مجلد المشروع** `english-learning-gh`.
(أسهل طريقة: افتح المجلد، اكتب `cmd` في شريط العنوان واضغط Enter)

### الخطوة 1: ثبّت الحزم (أول مرة فقط، تأخذ دقائق)
```
npm install
```

### الخطوة 2: شغّل التطبيق كتطبيق ويب (للمعاينة السريعة)
```
npm run dev
```
- سيعطيك رابط مثل `http://localhost:5173`
- افتحه في المتصفّح — هذا تطبيقك شغّال

### الخطوة 3: عاينه كتطبيق جوال على شاشة الكمبيوتر
- في المتصفّح (Chrome)، اضغط `F12`
- اضغط `Ctrl + Shift + M` (وضع الجوال)
- اختر "iPhone" من القائمة العلوية
- الآن تشوف تطبيقك بمقاس آيفون بالضبط، وتتنقّل فيه بالماوس

---

## 📦 تحويله لتطبيق iOS فعلي (لاحقاً — يحتاج ماك)

> ⚠️ هذي الخطوات تحتاج جهاز Mac أو خدمة سحابية. لا تشغّلها على ويندوز.
> نضبطها لما تجهز للنشر.

### إضافة منصّة iOS (مرّة واحدة، على ماك):
```
npm run cap:add:ios
```

### بناء ومزامنة وفتح Xcode (على ماك):
```
npm run cap:ios
```

---

## 🔧 ملاحظات مهمة

- **`npm run dev`** = معاينة سريعة كموقع (الأفضل للتطوير اليومي على ويندوز)
- **`npm run build`** = يبني نسخة الإنتاج في مجلد `dist`
- **`npm run cap:sync`** = يبني ويزامن مع تطبيق الجوال (يحتاج إضافة المنصّة أولاً)

- معرّف التطبيق الحالي: `com.englishpath.app` (في ملف `capacitor.config.ts`)
  - غيّره لمعرّفك الخاص قبل النشر النهائي على App Store

- اسم التطبيق: **مسار الإنجليزية**

---

## 🆘 لو واجهت مشكلة

- **"npm غير معروف"** → Node.js مو مثبّت صح، أعد تثبيته
- **خطأ أثناء install** → احذف مجلد `node_modules` وملف `package-lock.json` ثم أعد `npm install`
- **الموقع ما يفتح** → تأكّد أنك في المجلد الصحيح `english-learning-gh`

---

## 📋 الخطوة التالية بعد التشغيل

1. شغّل `npm run dev` وعايِن التطبيق
2. جرّبه في وضع الجوال (F12 ثم Ctrl+Shift+M)
3. لاحظ أي شي تبي تطوّره وأخبرني
4. نطوّر حبّة حبّة، وبعدها نجهّز نسخة iOS للنشر
